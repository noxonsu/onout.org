Features

Features

  • Lazy mint! Free minting! You don't pay anything when creating NFTs!
  • Based on top of rarible.org SDK, all new protocol updates will be adopted!
  • Users can buy NFTs using MetaMask
  • Set price for NFT in ETH or ERC20 tokens
  • ERC721, collectibles
  • Fully compatible with another marketplace, buyers can resell your NFTs on opensea.io, etc...
